# DevConnect

A real-time chat application where users can connect and send messages instantly. Built to understand how real-time communication works in web apps.

## Features

* Real-time messaging
* User authentication
* Live message updates without refresh
* Simple and clean UI

## Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: MongoDB
* Real-time: Socket.io

## How to Run

1. Clone the repo
2. Setup `.env` file
3. Start backend:
   npm run dev
4. Start frontend:
   npm start
